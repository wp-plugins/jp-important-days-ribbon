== Description ==

This plugin will automatically inform you to know the Bangladeshis Observance day.After activating this plugin it will inform you the Stated day.At the top of right corner of your site you see a Ribbon.There some of information of the mentioned day.You can arbitrary change your Ribbonn style from plugin option panel.

== Installation ==

<ul>
	<li>From your WordPress Dashboard Go for a "Add New" inside "Plugins" menu.</li>
	<li>Search for "JP Important Days Ribbon" and install the plugin.</li>
	<li>Activate the plugin</li>
</ul>

<strong>You:</strong> have finished your installation.<br>

Now go to admin panel from <strong>Dashboard > Settings > JP Important Days Ribbon Settings</strong> for Necessary settings.
